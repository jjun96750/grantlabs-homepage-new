# Security Policy

## Reporting

Report suspected security issues, exposed credentials, deployment mistakes, or contact-form abuse concerns to:

```text
jjun96750@gmail.com
```

Please include the affected URL, reproduction steps, observed behavior, and any relevant screenshots or response headers.

## Scope

This policy covers the static Grant Labs homepage in this repository, including:

- HTML/CSS/JavaScript source files
- Cloudflare Pages deployment configuration
- Static assets and metadata files
- Contact form configuration and fallback behavior

## Handling

Security reports should be reviewed before public disclosure. Do not publish exploit details, credentials, or private customer/contact information in GitHub issues or public comments.
